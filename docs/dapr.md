# Dapr development

## Prerequisites

- Dapr: 0.0.7
- Dapr CLI: 0.0.7
- Tye: 0.3.0-alpha.20265.1+ef441e87047930e04d24646e0d89183b07552747
- .NET 5 Preview 3

## Tye run

```bash
$ tye run tye-min.yaml
```

## Dapr run locally

```bash
$ powershell -f start.ps1
```

Waiting seconds, then you happy to go `http://localhost:5012`

Happy hacking!
